TeamSetup

@Immutable
data class TeamSetup(val name: String, val colorArgb: Long, val goalkeeperColorArgb: Long? = null, val formation: Formation, val kitStyle: TeamKitStyle = TeamKitStyle(), val attackDirection: AttackDirection = AttackDirection.LeftToRight)

High-level configuration for a single team in a match.

This allows you to define the team name, colours, formation, attack direction and kit style in one place and convert it to a TeamLineup when rendering.
